# My Virtual Pizza House
A space to explore PHP best practices  
...on the example of a pizza house.

## Kickstart
```shell
make up
make console

# inside console
composer install
```

## Tests
Inside console
```shell
make tests-unit
make tests-coverage
make tests-mutation
```

## Tools
Inside console
#### Rector instantly upgrades and refactors the PHP code of your application
https://github.com/rectorphp/rector
```shell
make rector
```
